FLUTTERGEDDON 1.0.1

EN:
- Added auto-update via GitHub Releases with SHA256 verification.
- Added CHANGELOG button with unread changes indicator.
- Added RU/EN language switcher. First launch follows Windows language, manual choice is saved.
- DLL switcher only works with user files already located in the Steam folder: version.dll/winhttp.dll are renamed to .disabled and back. The app does not copy its own DLL files into Steam.
- FLUTTERMODE, volume, language, and read changelog state are saved.

RU:
- Добавлено автообновление через GitHub Releases с проверкой SHA256.
- Добавлена кнопка CHANGELOG с индикатором непрочитанных изменений.
- Добавлен переключатель языка RU/EN, язык берется из Windows при первом запуске и потом запоминается.
- DLL-переключатель явно работает только с файлами пользователя в папке Steam: version.dll/winhttp.dll переименовываются в .disabled и обратно. Приложение не копирует свои DLL в Steam.
- Режим FLUTTERMODE, громкость, язык и прочитанный changelog сохраняются в настройках.
